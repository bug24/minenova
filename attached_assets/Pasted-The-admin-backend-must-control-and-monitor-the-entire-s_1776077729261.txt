The admin backend must control and monitor the entire system.

1. Admin Authentication System
Secure admin login with email and password
JWT authentication for admin sessions
Optional 2FA support (if possible)
Role support (future: super admin, support admin)
2. User Management

Admin must be able to:

View all registered users
Search users by email, username, or referral code
View individual user profile including:
wallet balance
mining status
referral stats
transaction history
Suspend or ban users
Reset user password (optional)
Manually adjust user balance (credit or debit)
3. Mining Control System

Admin must be able to:

View all active mining sessions
View mining rate per user
Start, stop, or reset mining for any user
Set global mining rate (USDT per minute or hour)
Set user-specific mining rate override
Disable mining system globally (maintenance mode)
4. Referral System Control

Admin must be able to:

Set referral bonus amount (default 0.25 USDT equivalent)
Set referral commission percentage (default 7%)
View all referral relationships
View referral earnings per user
Detect and flag suspicious referral activity
Disable referral system temporarily if needed
5. Wallet & Transaction System

Admin must be able to:

View all user wallets
See full transaction logs:
mining earnings
referral bonuses
referral commissions
withdrawals
manual adjustments
Filter transactions by type, user, or date
Manually credit or debit any wallet
Export transaction history (CSV or JSON)
6. Withdrawal Management System

Admin must be able to:

View all withdrawal requests
Filter by status (pending, approved, rejected)
Approve or reject withdrawal requests
Add admin notes to each withdrawal
Set minimum withdrawal amount
Track total platform withdrawals
Prevent duplicate withdrawal processing
7. Analytics Dashboard

Admin dashboard should show:

Total number of users
Active miners currently online
Total earnings generated on platform
Total referral payouts
Total withdrawals processed
Net platform balance (liability tracking)
Daily, weekly, monthly growth charts
8. System Settings Panel

Admin must be able to configure:

Mining rate (global)
Referral bonus value
Referral commission percentage
Minimum withdrawal amount
System maintenance mode toggle
Enable or disable referrals
Enable or disable mining system
9. Security & Anti-Fraud Controls
Prevent self-referrals
Detect multiple accounts from same device/IP (basic flagging)
Rate limit mining updates
Log all admin actions
Audit trail for changes made by admin
10. Tech Requirements
Backend: Node.js with Express
Database: MongoDB or PostgreSQL
Authentication: JWT
API structure: RESTful APIs
Must be modular and scalable
Clean separation of services (users, mining, wallet, referrals, admin)
